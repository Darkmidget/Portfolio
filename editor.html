Quick Reference — Adding a Project

  1. Fill in the form above with your project details
  2. Write your documentation in the Markdown editor
  3. Drop images into the drop zone to preview — then save them to resources/Pictures/your-project-id/
  4. Click "Generate Code" and copy the output
  5. Open src/data/projects.js and paste the entry into the projects array
  6. Commit & push to GitHub — done!
