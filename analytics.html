☁️ GitHub Integration (Project Popularity)

Track aggregate project popularity across all visitors by syncing anonymized view counts to a GitHub Gist. Your local visitor data stays private and never leaves their browser.

📋 How to set up GitHub sync (click to expand)
  1. Create a GitHub Gist: Go to https://gist.github.com, create a new gist named portfolio-analytics.json with empty content {}
  2. Get Gist ID: Copy the ID from the URL (after /gist.github.com/username/)
  3. Create Token: Go to GitHub Settings → Developer Settings → Personal Access Tokens → Generate new token. Grant gist scope only.
  4. Enter above: Paste your Gist ID and token, then click "Sync to GitHub"
  5. View results: After syncing, click "Fetch from GitHub" to see aggregated project popularity

Data Export & Privacy

📋 How This Works

Privacy-First Tracking: This portfolio collects anonymous analytics directly in your browser using localStorage. No data is sent to external servers. You have full control and can export or clear data anytime.

What's Tracked: Page views, project clicks, and search queries. Never tracked: IP addresses, user agents, cookies, or any personal information.

Data Storage: Stored in browser localStorage. Automatically cleaned up after 90 days. You can clear it anytime using the "Clear All Data" button.
